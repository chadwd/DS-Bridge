# Agent Sessions

Session plans for DS-Bridge contributors. Each session focuses on specific objectives with tracked tasks and success criteria.

## File Naming Convention

```
{contributor}-SESSION_{N}_PLAN.md
```

**Examples:**
- `chad-SESSION_8_PLAN.md` - Chad's Session 8 plan
- `jake-SESSION_2_PLAN.md` - Jake's Session 2 plan

## Session Plan Structure

Each plan includes:
- **Status**: IN PROGRESS, COMPLETE, or BLOCKED
- **Focus**: One-line summary of session goals
- **Objectives**: Prioritized list of goals
- **Tasks**: Detailed task lists with checkboxes
- **Success Criteria**: Measurable outcomes
- **Accomplishments**: Summary of what was done (added at completion)
